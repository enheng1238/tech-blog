# How Generative AI Works — 生成式AI到底是怎么工作的？

> 吴恩达《Generative AI for Everyone》第1章「How Generative AI Works」精读笔记

## AI 不是一块铁板，它是一个工具箱

吴恩达在课程中反复强调一个观点：**不要把 AI 当成一个单一的"东西"。它是一个工具箱，里面装着不同的工具，适用于不同的任务。**

目前最重要的两件工具是：

```
AI 工具箱
├── ① Supervised Learning（监督学习）—— 2010-2020 年代的主角
└── ② Generative AI（生成式AI）—— 正在崛起的新势力
```

了解这两者的关系，是理解 GenAI 工作原理的关键。

## 先理解 Supervised Learning

Supervised Learning 的概念极其简单：**给定输入 A，生成输出 B。**

```
输入 A → [Supervised Learning 模型] → 输出 B
```

吴恩达举了 7 个例子来展示这个模式有多通用：

| 场景 | 输入 A | 输出 B |
|------|--------|--------|
| 垃圾邮件过滤 | 一封邮件 | 是/否垃圾邮件 |
| 在线广告 | 用户信息 + 广告内容 | 是否点击 |
| 自动驾驶 | 摄像头画面 + 雷达数据 | 其他车辆的位置 |
| 医疗诊断 | X 光片 | 诊断结果 |
| 质检 | 产品照片 | 有无缺陷 |
| 语音识别 | 一段音频 | 文字转录 |
| 情感分析 | 一条评论 | 正面/负面 |

这个模式在 2010-2020 年间驱动了 AI 领域的大部分商业价值——从搜索引擎到推荐系统到广告投放。

> **为什么重要：** 吴恩达特意强调这段历史，因为 GenAI 本质上就是在 Supervised Learning 的根基上长出来的。不理解 Supervised Learning，就理解不了 GenAI 的能力边界。

## GenAI 的工作原理：超级简单的目标

那么，GenAI 是怎么用 Supervised Learning 做"生成"的呢？

答案是：**让模型预测下一个词。**

```
训练数据：互联网上海量的文本（书籍、网页、论文、代码...）

任务：给定前 N 个词，预测第 N+1 个词

例如：
  "法国的首都是____" → 模型需要输出"巴黎"
  "Python 是一种____" → 模型需要输出"编程语言"
  "先帝创业未半而____" → 模型需要输出"中道崩殂"
```

这个过程在数十亿个文本片段上反复进行。模型每次预测，拿自己的答案和真实文本对比，计算误差，反向传播修正参数。

**关键洞察：** 这个目标极其简单，但规模和数据量决定了结果。当模型在数千亿 token 上完成训练后，一个意料之外的现象发生了——它不再只是"预测下一个词"，而是涌现出**推理、类比、总结、翻译**等能力。

### 一个直观的理解方式

想象你是一个外星人，被扔进了一个装满全人类所有书籍、网页、聊天记录的房间里。你的任务只有一个：根据你读到的上一句话，猜下一句话是什么。

一开始你只会机械重复。但当你读了 1000 亿句话之后——你开始"理解"了语法、逻辑、因果关系、讽刺、隐喻。

你不是被直接教会了这些——你是在猜词的过程中自己"涌现"出来的。

这就是 GenAI 的工作原理。

## 一个数字说明一切

GPT-3 在约 **3000 亿 token** 上完成了训练。

做个对比：如果把一个人类从出生到 18 岁的所有阅读量加起来，大概在 **1-3 亿词**左右。GPT-3 的"阅读量"是人类一辈子的 **1000-3000 倍**。

> **为什么重要：** 理解这个"规模 × 简单目标 = 涌现能力"的模式，能帮你理解 GenAI 的强项和弱点。它不是在"理解"世界——它是在"看到"足够多的模式之后，产生了一种看起来像理解的行为。这让它在很多任务上表现出色，但也意味着它可能会在缺乏足够训练数据的边缘场景上"自信地胡说"。

## 从原理到实践

理解了工作原理后，吴恩达给出了一个实用的建议框架：

| 如果你需要 GenAI 做... | 可以信赖吗？ | 原因 |
|------------------------|------------|------|
| 写初稿、头脑风暴 | ✅ 非常可靠 | 模式匹配是它的强项 |
| 翻译、摘要 | ✅ 可靠 | 训练数据覆盖广 |
| 代码生成 | ⚠️ 需要检查 | 语法正确但可能有逻辑 bug |
| 精确计算 | ❌ 不可靠 | 它不是计算器 |
| 事实性知识 | ⚠️ 需要验证 | 可能有幻觉 |
| 实时信息 | ❌ 不可靠 | 训练数据有截止日期 |

这个表直接源于对"预测下一个词"这个工作机制的理解——它不是百科全书，它是一个基于统计模式的生成引擎。
